Eine Leaflet-Karte erstellen

Um eine Karte mit Leaflet auf einer Internetseite anzuzeigen, reichen wenige Schritte aus:

  • Integrieren Sie die notwendigen JavaScript und Cascading Style Sheet (CSS) Dateien.
  • Erstellen Sie ein <div>-Element in dem die Karte angezeigt werden soll.
  • Erstellen Sie das Karten-Objekt.
  • Fügen Sie einen Karten-Layer zur Karte hinzu.

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							</head>
							<body>
							</body>
							</html>
Integrieren Sie die notwendigen JavaScript und Cascading Style Sheet (CSS) Dateien

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							
							<link rel="stylesheet" href="https://unpkg.com/leaflet@1.1.0/dist/leaflet.css" />
							<script src="https://unpkg.com/leaflet@1.1.0/dist/leaflet.js"></script>
							
							</head>
							<body>
							</body>
							</html>
Erstellen Sie ein <div>-Element in dem die Karte angezeigt werden soll

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							<link rel="stylesheet" href="https://unpkg.com/leaflet@1.1.0/dist/leaflet.css" />
							<script src="https://unpkg.com/leaflet@1.1.0/dist/leaflet.js"></script>
							</head>
							<body>
							
							<div style="height: 180px;" id="mapid"></div>
							
							</body>
							</html>
Nun wird es spannend: Erstellen Sie das Karten-Objekt

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							<link rel="stylesheet" href="https://unpkg.com/leaflet@1.1.0/dist/leaflet.css" />
							<script src="https://unpkg.com/leaflet@1.1.0/dist/leaflet.js"></script>
							</head>
							<body>
							<div style="height: 180px;" id="mapid"></div>
							
							<script>
							var mymap = L.map('mapid').setView([50.27264, 7.26469], 13);
							</script></body>
							
							</html>
Fügen Sie einen Karten-Layer zur Karte hinzu

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							<link rel="stylesheet" href="https://unpkg.com/leaflet@1.1.0/dist/leaflet.css" />
							<script src="https://unpkg.com/leaflet@1.1.0/dist/leaflet.js"></script>
							</head>
							<body>
							<div style="height: 180px;" id="mapid"></div>
							<script>
							var mymap = L.map('mapid').setView([50.27264, 7.26469], 13);
							
							L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png').addTo(mymap);
							
							</script>
							</body>
							</html>

Stamen (https://stamen.com/)

							<!DOCTYPE HTML>
							<html>
							<head>
							<title>Eine OSM Karte mit Leaflet</title>
							<link rel="stylesheet" href="https://unpkg.com/leaflet@1.1.0/dist/leaflet.css" />
							<script src="https://unpkg.com/leaflet@1.1.0/dist/leaflet.js"></script>
							
							<script type="text/javascript" src="http://maps.stamen.com/js/tile.stamen.js"></script>
							
							</head>
							<body>
							<div style="height: 180px;" id="mapid"></div>
							<script>
							var mymap = L.map('mapid').setView([50.27264, 7.26469], 13);
							
							L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png').addTo(mymap);
							
							
							var layer = new L.StamenTileLayer("watercolor");
							mymap.addLayer(layer);
							
							</script>
							</body>
							</html>
